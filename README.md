## <a name="setup"></a> Getting Started

### Install Yarn via Homebrew (macOs)

```sh
$ brew install yarn
```

### Install package dependencies

```sh
$ yarn
```

### Copy .env.example to .env

```sh
$ cp .env.example .env
```

### Start docker dependencies

```
$ docker-compose up -d
```

### Run the application

```sh
$ yarn dev
```
